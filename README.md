# Binarycr

Trading bot for Binary.com

## Installation

```crystal
crystal build src/binarycr.cr --release
```

## Usage

```bash
./binarycr --token=xxx --application=xxx --trade_amount=10 --wanted_profit=100 --stop_loss=256 --alternate
```
